할 일 입력 가이드

아래 가이드를 본인의 AI에게 전달하면 Cowork에 맞는 형식으로 정리할 수 있습니다.

  • 서로 독립적인 작업은 각각 별도 티켓으로 나뉩니다.
  • 작업 내용과 완료 조건은 해당 티켓의 상세 내용으로 들어갑니다.
  • 회의·일정·단순 메모·완료한 작업은 제외됩니다.
다음 내용을 Cowork에 붙여넣을 수 있는 작업 목록으로 정리해줘.

규칙:
1. 서로 독립적으로 실행할 작업은 각각 별도 항목으로 작성한다.
2. 각 작업은 아래 형식을 사용한다.

제목: 작업 제목
작업 내용:
- 수행할 내용
완료 조건:
- 확인 가능한 완료 기준

3. 회의, 일정, 단순 메모, 이미 완료한 작업은 제외한다.
4. 입력에 없는 사실이나 완료 조건을 새로 만들지 않는다.
5. 마크다운 코드 블록이나 추가 설명 없이 정리된 작업 목록만 출력한다.
